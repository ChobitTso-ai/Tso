# 🎮 Minecraft 測驗遊戲

一個使用 Minecraft 主題設計的桌面測驗小遊戲，讓小孩能夠透過遊戲化的方式複習功課！

## ✨ 功能特色

- 🎨 **Minecraft 主題設計**：使用 Minecraft 經典配色和風格
- 📤 **靈活的題目管理**：支援上傳檔案或手動輸入題目
- 🎲 **隨機出題**：可設定題目範圍，系統會隨機出題
- 💯 **即時評分**：答題後立即顯示對錯和正確答案
- 💾 **自動儲存**：題目會自動儲存，下次開啟時自動載入
- 🖥️ **完全離線**：不需要網路連線或架設伺服器

## 🚀 如何使用

### 安裝需求

- Python 3.6 或更新版本
- Tkinter（通常 Python 已內建）

### 執行遊戲

1. **Windows 使用者**：
   ```bash
   python minecraft_quiz.py
   ```

2. **Mac/Linux 使用者**：
   ```bash
   python3 minecraft_quiz.py
   ```

### 使用步驟

1. **新增題目**：
   - 點擊「📤 上傳題目檔案」上傳 TXT 或 CSV 檔案
   - 或點擊「✏️ 手動輸入題目」直接輸入題目

2. **設定範圍**：
   - 在「出題範圍」區域設定要出題的範圍
   - 例如：從第 1 題到第 10 題

3. **開始測驗**：
   - 點擊「🎯 開始測驗！」按鈕
   - 系統會隨機從設定範圍內抽題

4. **答題**：
   - 在答案欄位輸入答案
   - 按「✅ 提交答案」或直接按 Enter 鍵
   - 查看對錯後點擊「➡️ 下一題」繼續

## 📝 題目格式

題目檔案必須使用以下格式（每行一題）：

```
問題|答案
```

### 範例

```
1+1=?|2
2+2=?|4
3x3=?|9
首都是台北的國家是？|台灣
What is the capital of France?|Paris
```

## 📂 檔案說明

- `minecraft_quiz.py` - 主程式檔案
- `questions.json` - 自動產生的題目儲存檔案
- `example_questions.txt` - 範例題目檔案（數學題）
- `example_questions_chinese.txt` - 範例題目檔案（國語題）

## 🎯 功能說明

### 控制面板

- **📤 上傳題目檔案**：選擇並上傳包含題目的文字檔案
- **✏️ 手動輸入題目**：開啟輸入視窗，直接輸入題目
- **📋 檢視所有題目**：查看目前題庫中的所有題目
- **🗑️ 清空題庫**：刪除所有題目（在檢視題目視窗中）

### 測驗區域

- **出題範圍**：設定要從第幾題到第幾題進行測驗
- **🎯 開始測驗**：開始新的測驗回合
- **💎 分數**：顯示目前答對的題數
- **📚 題庫**：顯示題庫中的總題數

## 💡 使用技巧

1. **分段練習**：可以設定小範圍（例如 1-10 題）進行集中練習
2. **重複練習**：同一個範圍可以重複測驗，每次都會隨機出題
3. **題目分類**：可以建立不同的題目檔案（如數學、國語、英文），需要時再上傳
4. **即時回饋**：答錯時會立即顯示正確答案，幫助記憶

## 🎨 介面配色

遊戲使用 Minecraft 經典配色：
- 🟩 草地綠色（標題區）
- 🟫 泥土棕色（功能按鈕）
- ⬜ 石頭灰色（控制面板）
- 💎 鑽石藍色（開始測驗按鈕）
- ⭐ 金色（分數顯示）

## ⚠️ 注意事項

- 題目和答案請使用 UTF-8 編碼
- 題目檔案中每行必須包含「|」符號分隔問題和答案
- 答案比對時不區分大小寫
- 題目會自動儲存在 `questions.json` 檔案中

## 🐛 常見問題

**Q: 為什麼我上傳的題目沒有載入？**
A: 請確認檔案格式正確，每行都是「問題|答案」的格式，且使用 UTF-8 編碼。

**Q: 可以編輯已經上傳的題目嗎？**
A: 可以點擊「檢視所有題目」查看，如需修改可以清空題庫後重新上傳。

**Q: 題目會保存嗎？**
A: 會的！所有題目都會自動儲存在 `questions.json` 檔案中，下次開啟時會自動載入。
